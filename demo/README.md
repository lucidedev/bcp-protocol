# BCP Interactive Demo

An interactive visualization of the BCP negotiation flow between a buyer and seller agent.

## What it shows

- Two company profiles (buyer & seller) negotiating a B2B deal
- Real-time message flow: Intent → Quote → Counter → Commit → Fulfil
- x402 payment panel with on-chain escrow status and transaction hashes
- State machine transitions as the deal progresses

## Running locally

```bash
cd demo
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
